docker-compose.yml и документ с перечислением используемых 
образов лежат в папке Docker

Чтобы запустить локально:

1) В папке Docker выполнить команду "docker-compose up"
	Api будет доступен по адресу "http://localhost:5000"
	Так же будет создана тестовая база.

2) Чтобы запустить клиент - выполнить комманду:
 "docker run --name client -p 4200:80 feeshka/client-nw"

Но клиент при переходе по любой ссылке почему-то падает с ошибкой 404 (nginx/1.19.8). 
Возможно что-то в используемом базовом образе. Не удалось разобраться в чём проблема.

Поэтому клиента лучше запускать через vsCode из ветки:
"features/docker". Так всё работает корректно.
